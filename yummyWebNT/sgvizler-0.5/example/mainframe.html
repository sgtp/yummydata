Sgvizler Examples

Many live examples of using Sgvizler are gathered here. Each example page contains a chart drawn by Sgvizler on page load (some can take a while, so be patient, i.e, a few seconds), a listing of the <div> element which specifies the chart, and the query results in a table. Additionally, some pages may also contain other charts visualizing the exact same query results.

If you're not seeing this page as frames there is a frames version which might be more handy.

Endpoints

NPD
Linked Open NPD Fact-pages (endpoint http://sws.ifi.uio.no/sparql/world) contains data collected from the Norwegian Petroleum Directorate's (NPD) "FactPages", which is a database about the activity on the Norwegian Continental Shelf, e.g. oil fields, wellbores, drilling and production.
DBpedia
dbpedia.org (endpoint http://dbpedia.org/sparql) is probably well-known to you since you read this. Sometimes the dbpedia endpoint is slow to respond.
World
The "world" database (endpoint http://sws.ifi.uio.no/sparql/world) contains data about the cities, countries, continents, and so on, of the world. The data is not up do date, but it serves well as an example database for this purpose. The data source for this dataset is a world database which ships with MySQL.
